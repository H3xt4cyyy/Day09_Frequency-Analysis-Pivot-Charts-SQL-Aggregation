# Data Analytics Level III - Day 9: Data Summarization & Optimization

## 📊 Executive Overview
This repository contains the data modeling, visualization, and SQL optimization deliverables for the Day 9 Data Analytics module. The primary objective of this project is to summarize transactional datasets (Core 3) through frequency analysis, pivot charts, and advanced SQL aggregations. 

The datasets processed herein simulate a retail environment requiring regional performance tracking, customer segmentation, and database query optimization.
